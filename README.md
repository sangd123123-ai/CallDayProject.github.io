## ✨ 프로젝트 소개

<div align="center">

![프로젝트 메인](/ppt/프로젝트%20메인.JPG)

</div>

> **CallDayProject**는 운동을 시작하고 꾸준히 지속하고 싶은 모든 사람들을 위한 종합 피트니스 관리 플랫폼입니다.

### 🎯 프로젝트 목표

- 📱 **직관적인 사용자 경험**: 누구나 쉽게 사용할 수 있는 인터페이스
- 📈 **체계적인 관리**: 운동 계획, 실행, 기록을 한 곳에서
- 🤝 **커뮤니티 활성화**: 피드 공유로 동기부여 극대화
- 🤖 **AI 기반 추천**: 개인 맞춤형 식단 추천 시스템
